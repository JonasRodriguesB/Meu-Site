Cotação de Moeda

DESCRIÇÃO

Este é meu segundo projeto em Django e agora meu objetivo foi conseguir realizar a integração da aplicação em Django com uma API externa, podendo manipular os dados em JSON e apresentá-los ao usuário. O projeto apresentado aqui é a versão original dele, eu ainda vou adicionar outras funcionalidades que foram previstas quando eu desenhei o projeto.

O projeto consiste em uma página de visualização em tempo real da cotação de diferentes moedas. A cotação de todas moedas é convertida para o R$ (BRL) e selecionei apenas 3 moedas para o projeto inicial. A imagem a seguir, da página inicial, mostra a cotação do dólar, do euro e do bitcoin. Os valores são apresentados em três blocos, sendo o bloco maior referente ao valor do dólar e os dois blocos menores referentes aos valores do euro (esquerda) e do bitcoin (direita).

Imagem da Página do Projeto

Abaixo é possível ver o horário em que foi realizada a última atualização da API e a data, assim a pessoa consegue ter certeza de quão atualizado está o valor. Ainda acrescentei um anel de carregamento para que o usuário possa visualizar quando a página deve ser carregada. A ideia do anel era compôr uma funcionalidade de auto-carregamento da página, mas eu não fiquei satisfeito com o resultado e essa funcionalidade deve ser implementada no futuro. O anel ficou, porque agora ele serve como um contador simples de quão atualizado estão os valores.

A API fornece dados atualizados a cada 30~40 segundos, então configurei o anel de carregamento para um período de um minuto. Quando o anel se completa, é apresentada uma informação na página do usuário sugerindo que ele atualize a página para que os valores possam ser atualizados também, assim o usuário pode carregar a página apenas quando quiser ver os valores atualizados.
